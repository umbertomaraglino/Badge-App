
# Badge-App

Badge-App è una semplice applicazione Flutter per segnare l'orario di ingresso e uscita dall'ufficio, calcolare i minuti di straordinario/settimana e visualizzare la lista delle timbrature.

## Funzionalità principali
- Segna ingresso e uscita con un click
- Visualizza la lista delle timbrature
- Modifica manuale degli orari
- Calcolo automatico dei minuti settimanali
- Reset manuale e automatico ogni lunedì
- Salvataggio locale dei dati

## Come funziona
1. Premi "Segna Ingresso" all'arrivo in ufficio.
2. Premi "Segna Uscita" quando esci.
3. Puoi modificare manualmente gli orari cliccando sulla riga.
4. Il totale dei minuti della settimana viene calcolato automaticamente.
5. Puoi resettare la settimana manualmente o attendere il reset automatico ogni lunedì.

## Avvio rapido

```bash
flutter pub get
flutter run
```
